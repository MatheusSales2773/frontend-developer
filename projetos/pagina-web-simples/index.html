Data Science: Transformando Dados em Decisões

Pilares Fundamentais

Machine Learning

É o coração da Ciência de Dados. Consiste em algoritmos que permitem que os computadores "aprendam" a partir dos dados, identificando padrões e fazendo previsões.

Big Data

Refere-se a conjuntos de dados extremamente grandes e complexos. É definido pelos "3 Vs": Volume, Velocidade e Variedade.

  • Volume: Refere-se à quantidade de dados gerados a cada segundo.
  • Velocidade: Refere-se à rapidez com que os dados são gerados e processados.
  • Variedade: Refere-se à diversidade dos tipos de dados (estruturados, semiestruturados, não estruturados).

Estatística e Matemática

São a base teórica. Conceitos como probabilidade, testes de hipótese e regressão são fundamentais para validar os resultados e garantir a confiabilidade das conclusões.
